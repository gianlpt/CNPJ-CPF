Validador e Gerador de CPF e CNPJ

Aplicação web desenvolvida em React com Vite para validar e gerar documentos brasileiros.

O projeto permite validar automaticamente CPF, CNPJ numérico e CNPJ alfanumérico em um único campo. Também permite gerar documentos válidos com ou sem máscara.

Funcionalidades
Validação automática de CPF
Validação automática de CNPJ numérico
Validação automática de CNPJ alfanumérico
Geração de CPF válido
Geração de CNPJ numérico válido
Geração de CNPJ alfanumérico válido
Opção para gerar documentos com máscara
Opção para gerar documentos sem máscara
Cópia rápida do documento gerado
Tecnologias utilizadas
React
Vite
JavaScript
CSS


Observação

Este projeto tem finalidade educacional e de apoio ao desenvolvimento/testes. Os documentos gerados são válidos em regra de dígito verificador, mas não representam necessariamente documentos reais de pessoas ou empresas.

Versão online: https://cnpj-cpf.vercel.app/
